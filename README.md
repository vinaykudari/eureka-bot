# eureka-bot
### Software
- Cross Platform Mobile App (Flutter) at `/app`
- Server at `/server`

### Hardware
- Jetson Nano v2
- 15D Dual Channel DC Motor Driver
- VL53L0X - ToF Sensor
- APDS9960 - Proximity Sensor
